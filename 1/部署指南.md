# 🎮 贪吃蛇游戏部署指南

## 🚀 方法一：GitHub Pages（免费，推荐）

### 步骤1：创建GitHub仓库
1. 访问 [GitHub.com](https://github.com) 并登录
2. 点击右上角 "+" 按钮，选择 "New repository"
3. 仓库名称：`snake-game` （或你喜欢的名字）
4. 选择 "Public"（公开）
5. 点击 "Create repository"

### 步骤2：上传游戏文件
1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 将以下三个文件拖拽到上传区域：
   - `index.html`
   - `script.js`
   - `style.css`
3. 在页面底部添加提交信息：`Add snake game files`
4. 点击 "Commit changes"

### 步骤3：启用GitHub Pages
1. 进入仓库的 "Settings" 标签
2. 滚动到左侧 "Pages" 选项
3. 在 "Source" 下选择 "Deploy from a branch"
4. 选择 "main" 分支和 "/ (root)" 文件夹
5. 点击 "Save"
6. 等待几分钟，页面会提供一个链接，格式类似：
   `https://你的用户名.github.io/snake-game/`

### 步骤4：分享给朋友
- 将生成的链接发送给朋友即可！
- 任何有链接的人都能直接玩你的游戏

## 🌐 方法二：Netlify（免费）

### 步骤1：访问Netlify
1. 打开 [Netlify.com](https://netlify.com)
2. 点击 "Sign up" 使用GitHub账户登录

### 步骤2：部署游戏
1. 登录后，点击 "Add new site" → "Deploy manually"
2. 将整个项目文件夹拖拽到部署区域
3. 等待部署完成，Netlify会提供一个链接

### 步骤3：分享
- 复制Netlify生成的链接分享给朋友

---

## 📱 方法三：CodePen（适合分享给开发者朋友）

### 步骤1：创建新的Pen
1. 访问 [CodePen.io](https://codepen.io)
2. 注册或登录账户
3. 点击 "Create" → "Pen"

### 步骤2：添加代码
1. **HTML部分**：复制 `index.html` 中的游戏HTML内容
2. **CSS部分**：复制 `style.css` 的内容到CSS面板
3. **JavaScript部分**：复制 `script.js` 的内容到JS面板

### 步骤3：保存和分享
1. 点击 "Save" 保存Pen
2. 复制链接分享给朋友

---

## 💻 方法四：简单本地服务器分享

### 使用Python内置服务器
如果你的朋友也在同一局域网内：

```bash
# 在游戏文件夹中打开命令行
python -m http.server 8000

# 然后告诉朋友访问：
# http://你的IP地址:8000
```

### 查看本机IP地址
```bash
# Windows
ipconfig

# macOS/Linux
ifconfig
```
